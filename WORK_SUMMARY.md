## 已完成工作

1. 新建项目
   - 在 D 盘创建项目目录 `D:\douyin-weight-analyzer`
   - 搭建 Next.js + React + Tailwind CSS 项目
   - 配置 TypeScript、ECharts、Vitest 等基础依赖

2. 核心功能
   - 支持输入抖音主页链接进行账号分析
   - 支持自动发现同赛道对标账号
   - 支持账号评分、权重等级、数据完整度、置信度展示
   - 支持雷达图能力展示
   - 支持对标差距分析
   - 支持运营建议生成
   - 支持发布时间建议
   - 支持流量池预测、星级评级、账号价值估算

3. 账号分析逻辑
   - 评分维度包含账号基础、更新稳定、互动效率、内容垂直、爆款能力、风险信号
   - 运营建议会根据账号实际垂直领域动态生成
   - 不再写死某个领域，例如光遇账号会输出光遇方向，其他账号会按对应领域分析
   - 发布时间建议优先参考主账号真实高表现时段，再用对标账号做轻量校准

4. 示例账号适配
   - 已支持景羽账号短链识别：`https://v.douyin.com/4e-eS02n5_M/`
   - 可展示昵称、头像、简介、粉丝数、获赞数、作品数等信息
   - 针对该账号输出垂直领域分析、发布时间建议和运营建议

5. UI 设计调整
